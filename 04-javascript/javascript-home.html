Introdução

JavaScript é uma linguagem de programação que adiciona interatividade ao seu site.

Cada vez que uma página da web faz mais do que apenas ficar ali e exibir informações estáticas para você ver - exibindo atualizações de conteúdo oportunas, mapas interativos, gráficos animados 2D/3D ou mais - você pode apostar que provavelmente o JavaScript está envolvido.
