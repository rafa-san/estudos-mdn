HTML - Apresentação

HTML (HyperText Markup Language) é uma linguagem de marcação usada para estruturar uma página web e seu conteúdo. O conteúdo pode ser estruturado em um conjunto de parágrafos, lista de marcadores, imagens ou tabelas de dados.

HTML consiste em uma série de elementos (tags) que você usa para delimitar, agrupar ou marcar diferentes partes do conteúdo para que apareçam ou atuem de uma determinada forma.

Uma tag HTML pode, por exemplo, criar um link que direcionará a página atual para uma outra página; ou colocar palavras em itálico; ou então incluir uma imagem dentro da página e assim por diante.
